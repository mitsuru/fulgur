Link Stylesheet Example

This example demonstrates loading CSS via <link rel="stylesheet" href="theme.css"> instead of the --css CLI flag.

How It Works

Automatic Base Path

When you pass an HTML file to the CLI, fulgur automatically sets the base path to the file's directory. Relative href values in <link> tags are resolved from there.

Local Files Only

Following fulgur's offline-first design, only local file paths are resolved. URLs starting with http:// or https:// are ignored.

Multiple Stylesheets

You can include multiple <link> tags. Each referenced CSS file is loaded and applied in order.
