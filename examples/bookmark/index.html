CSS Bookmark Properties

This example demonstrates the GCPM bookmark-level and bookmark-label properties. Open the PDF in a viewer that shows the outline (bookmarks) panel to see the result.

Automatic Headings

Headings h1 and h2 automatically appear in the PDF outline via fulgur's built-in UA stylesheet. Each heading gets a bookmark at its corresponding level with its text content as the label.

Suppressed Subheading

This h3 has bookmark-level: none in the author stylesheet, so it does not appear in the outline — even though fulgur's UA stylesheet would normally assign it level 3.

Custom Elements

Any element can opt into the outline. The appendix sections below use bookmark-level: 2 and bookmark-label: "Appendix: " attr(data-title) to create outline entries from a data-title attribute.

Appendix A — Glossary

bookmark-level: Sets the nesting depth of an outline entry. An integer (1–6 typical) or none to suppress.

bookmark-label: Defines the text shown in the outline. Accepts content(), attr(), and string literals.
