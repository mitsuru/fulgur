CSS Multi-column Layout

fulgur routes CSS multicol containers through a custom Taffy layout hook. Taffy sees the balanced column height as the container's intrinsic size, so surrounding siblings flow naturally around it — no post-pass reshape, no sibling overlap.

Two columns, four paragraphs

Alpha paragraph. Each child block is measured at the column width and then packed into balanced columns.

Beta paragraph. The balance algorithm searches for the smallest per-column budget that fits all children without overflow.

Gamma paragraph. When the multicol grows or shrinks relative to Taffy's initial pass, siblings after it are shifted to match.

Delta paragraph. Two-column layout is the canonical case for this hook design.
